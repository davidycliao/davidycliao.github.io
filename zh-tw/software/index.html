程式碼


flaiR R

R 語言封裝套件，用於在 R 中存取 Python 的 flair NLP 函式庫。Flair NLP 以其獨特的字元級別與上下文感知嵌入著稱，支援 BERT、ELMo、RoBERTa 等多種模型架構，並可存取 Hugging Face 上的最新模型。

legisTaiwan R

R 套件，透過台灣立法院 API 即時下載立法資料，包括院會發言紀錄、法案提案人與連署人資訊等。

asmcjr R

asmcjr 為教科書《Analyzing Spatial Models of Choice and Judgment, 第二版》的配套套件，提供簡化的空間縮放與 ggplot2 視覺化函式。

legisCrawler Python

Python 自動化網路爬蟲框架，用於擷取立法院網站上的國會質詢資料。

bisCrawler Python

Python 自動化網路爬蟲框架，用於擷取國際清算銀行（BIS）網站上的央行行長演講資料。
