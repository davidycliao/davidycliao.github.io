Teaching Repositories

DMASS (with Shih-chan Dai, Spring Term 2021, University of Massachusetts-Amherst)

The repository stores lesson files for “Data Management and Analysis for Social Sciences", University of Massachusetts-Amherst. This course aims to provide students with an overview of the theoretical and practical foundations required for managing and analyzing data, especially those in social sciences.

ESS3L (with Royce Carroll, 2020 53rd Essex Summer School in Social Science Data Analysis)

The repository stores lesson files for “3L Scaling Methods for Social Science: Estimating Patterns and Preferences in Surveys and Behavior", taught by Royce Carroll.



Packages

asmcjr (with Dave Armstrong, Royce Carroll, Christopher Hare, Ryan Bakker, Keith Poole, Howard Rosenthal)

The asmcjr package supports the book “2nd Analyzing Spatial Models of Choice and Judgment”, and streamlined functions for scaling and visualising estimates with ggplot2.

legisTaiwan

This is R interface package that facilitates access to the Taiwan Legislative Database. The dataset includes legislator’s voting decisions, political and sociodemographics. Requesting granting of authorization for public-use data via this package.



Resources & Links

UDpipe Linguistic Models

UDPipe is a trainable pipeline for tokenization, tagging, lemmatization and dependency parsing of CoNLL-U files. UDPipe is language-agnostic and can be trained given annotated data in CoNLL-U format. Trained models are provided for nearly all UD treebanks. Third-party R CRAN package is hosted by @jwijffels .

NLP Pipeline in Spacy and its Language Models

The language models provide multiple tasks for processing linguistic features such as text processing, dependency, POS tagging, named entities as well as a portal to train/improve existing language model for your own purpose.

ParlSpeech Database

ParlSpeech V2 updates more than 6.3 million parliamentary speeches in the key legislative chambers of Austria, the Czech Republic, Germany, Denmark, the Netherlands, New Zealand, Spain, Sweden, and the United Kingdom, covering periods between 21 and 32 years. The database is built by Christian Rauh and Jan Schwalbach . Please see @yl17124 if you would like to know what NLP techniques can be applied to this dataset.

The Centre of Taiwan Legislative Studies, Soochow University

The research centre inventories legislative roll calls from 2nd session to 8th in the Legislative Yuan (立法院), Taiwan. Plesee check @yl17124 if you would like to know what ideal point estimates can be applied to this dataset.
