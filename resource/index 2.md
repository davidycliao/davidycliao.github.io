### Resources & Links
</div>


**UDpipe Linguistic Models** 

<div style="text-align: justify">
  
UDPipe is a trainable pipeline for tokenization, tagging, lemmatization and dependency parsing of CoNLL-U files. UDPipe is language-agnostic and can be trained given annotated data in CoNLL-U format. Trained models are provided for nearly all UD treebanks. Third-party R CRAN package is hosted by [@jwijffels](https://github.com/bnosac/udpipe.models.ud). 

</div>

----

**NLP Pipeline in Spacy and its Language Models**

<div style="text-align: justify">  

The language models provide multiple tasks for processing linguistic features such as text processing, dependency, POS tagging, named entities as well as a portal to train/improve existing language model for your own purpose. 

----

**ParlSpeech Database** 
  
<div style="text-align: justify">

[ParlSpeech V2](https://dataverse.harvard.edu/dataverse/ParlSpeech) updates more than 6.3 million parliamentary speeches in the key legislative chambers of Austria, the Czech Republic, Germany, Denmark, the Netherlands, New Zealand, Spain, Sweden, and the United Kingdom, covering periods between 21 and 32 years. The database is built by [Christian Rauh](https://twitter.com/ChRauh) and [Jan Schwalbach](https://twitter.com/JanSchwalbach). Please see [@yl17124](https://yl17124.github.io/Parliament-Speech-Processing/) if you would like to know what NLP techniques can be applied to this dataset. 

</div>

----


**The Centre of Taiwan Legislative Studies, Soochow University**
<div style="text-align: justify">
  
  The research centre inventories legislative roll calls from 2nd session to 8th in the Legislative Yuan (立法院), Taiwan.  Plesee check @yl17124 if you would like to know what ideal point estimates  can be applied to this dataset.
